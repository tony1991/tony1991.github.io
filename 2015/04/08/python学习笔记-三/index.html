Python与java的区别

最后，结合我这几天的亲身感受，我列出了以下几点python与java的区别（不全还望见谅，欢迎补充）：

1.代码书写格式

  • JAVA 里的代码块用大括号对包括，Python 以冒号 + 四个空格缩进表示。why 空格而非tab?
    不仅仅是python，对于编程来说用space替代tab不管对于哪种语言都是一个好的选择，因为代码文件可能会在不同的环境、用不同的编辑器打开，而对于space的处理几乎所有的编辑器、所有的OS环境都是一样的，而对于tab的处理却不尽相同，有的会直接展开成空格，有的不会，而且展开为空格的话，有的是4个有的是8个，这样会造成代码的格式看起来不一致。对于python来说这个问题更加重要，因为缩进对于python来说是语法的一部分，所以将tab弄成space会减少很多麻烦。
  • JAVA 每行语句以分号结束，Python 可以不写分号。
  • JAVA 中的字符串以双引号括起来，Python 中单引号或双引号都可以（与 JavaScript 一样）。
  • JAVA 的类型要声明，Python 的类型不需要。
  • JAVA 基本上是类/结构操作，也就是面向对象处理，Python 可以以独立的函数模块来处理逻辑而不需要放到类中。

    2.类

  • python的类构造函数带参的话，第一个永远是self，而java不用。这里的self类似于java的this
  • 类继承：Java单继承，python采用mixin设计，可以多重继承。
  • 类属性定义：python里，以单下划线开头_A的表示不能直接访问，可通过类提供的接口访问；变量的名称前加上双下划线__B,代表私有private，只有类对象自己可以访问；
